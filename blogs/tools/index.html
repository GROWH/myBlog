    搭建个人博客

    zhaohuan 5/20/2020 工具

    此次搭建个人博客使用的是 vuepress-theme-reco + github 实现的

    部署 VuePress 到 GitHub Pages (超详细)：https://juejin.cn/post/6844904122873806856 使用vuepress搭建GitHub pages静态博客页面：https://www.cnblogs.com/zjjDaily/p/13633786.html https://www.cnblogs.com/LHLVS/p/11376732.html， https://www.cnblogs.com/LHLVS/p/11913940.html， https://juejin.cn/post/6844904122873806856#heading-2， https://segmentfault.com/a/1190000017890986， https://blog.csdn.net/weixin_33800463/article/details/88674024#t0，

    前言： 搭建个人博客的框架有很多，例如hexo、vuepress、jekyll等 以下是vuepress的官网链接：https://www.vuepress.cn/(opens new window)

    # 一、准备工作

    在开始之前，先下载好需要的一些东西，例如：node、git、vuepress node下载链接：http://nodejs.cn/download/ git下载链接:https://git-scm.com/downloads

    # 二、安装脚手架

    可以使用vuepress，但是vuepress本身是一个文档型的页面，比较简约，所以这里推荐一个大佬制作好的主题vuepress-theme-reco,其页面效果大概是这样的：图片1 随便找个路径，在终端打开，输入以下指令，安装全局vuepress-theme-reco： npm install @vuepress-reco/theme-cli -g

    等待下载好之后，，在输入以下指令进行初始化项目(blog是项目名字，自已也可以随便起个)： theme-cli init blog

    在初始化时，会让你填写一些信息，例如：博客名称、博客标题、博客描述等等 你可以先随便写点文字，后续这些信息都可以改的 等项目全部初始化完了后，文件目录是这样的：图片2

    了解完文件目录结构之后，我们就开始启动项目，因为刚生成的项目很多依赖包还没安装，并且当前的路径也没在项目中，所以依次执行以下指令： //第一步，进入到项目根目录 cd blog

    //第二步，安装依赖包 npm install

    //第三步，等依赖包下载完了以后在执行该指令启动项目 npm run dev

    启动好以后会自动打开网页，我们可以看到这样的画面了：图片3

    # (3) dest

    该属性表示的是项目打包以后文件生成的目录

    默认是生成 public文件夹，假如我们将dest的值改为./dest，那么当我们执行npm run build打包文件后，会在当前根目录下生成一个dist文件夹，打包好的项目都配置在该文件夹中。

    # (4) head

    该属性表示的是html文档的head标签中需要额外插入的其他标签 默认生成的是

      {
        "rel": "icon",    // rel属性值为icon
        "href": "/favicon.ico"   // href属性值为 /favicon.ico
      }
    ],
    [
      "meta",      //生成meta标签
      { 
        "name": "viewport",   // name属性值为viewport
        "content": "width=device-width,initial-scale=1,user-scalable=no"  // content属性值为width=device-width,initial-scale=1,user-scalable=no
      }
    ]

    # 五、添加评论模块

    其实这个主题框架已经内置了评论插件@vuepress-reco/vuepress-plugin-comments，我们需要做的就是做一些额外的配置

    首先去Valine官网 (opens new window)https://console.leancloud.cn/register 注册一下Valine账号，注册好后进行登录， 需要实名认证， 然后进入控制台，点击创建应用， 应用创建好以后，进入刚刚创建的应用，选择左下角的设置>应用凭证，然后就能看到你的APP ID和APP Key了：

    拿到APP ID和APP Key后，我们来到我们项目中.vuepress/config.js中的themeConfig属性中，按如下设置即可：

    module.exports = {
  // ...
  themeConfig: {
    // ....
    valineConfig: {     // valine 评论功能配置信息
      appId: '在这输入你的appid',// your appId
      appKey: '在这输入你的appKey', // your appKey
      placeholder: '尽情留下你想说的话吧~',           // 评论框占位符
      avatar: 'wavatar',           // 评论用户的头像类型
      highlight: true,         // 代码高亮
      recordIP: true,         // 记录评论者的IP
    },
  }
  // ...
}
